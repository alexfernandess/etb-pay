O Projeto

A Teoria Encontra a Produção

Desenvolvido de forma colaborativa pelos alunos da Escola Técnica de Brasília (ETB), o ETB-PAY é um sistema financeiro simulado construído totalmente do zero. Nosso objetivo não é apenas escrever código, mas formar profissionais com raciocínio crítico e visão sistêmica.

Aqui, traduzimos a teoria seca exigida nos editais para a prática implacável do mercado. Você aprenderá a projetar arquiteturas escaláveis, garantir a integridade dos dados e gerenciar o ciclo de vida do software sob os frameworks de governança mais rígidos do setor público e privado.

metodologia.json 
const doubleGain = {
  objetivo: "Aprovação e Empregabilidade",
  pilares: [
    "Código Blindado e Arquitetura Limpa",
    "Mapeamento de Bancas (Cebraspe, FGV, FCC...)",
    "Governança Corporativa e Agilidade"
  ],
  executar: function() {
    return this.pilares.map(p => p.aplicar());
  }
};

A Estrutura do Ecossistema

O projeto é fragmentado e integrado por três frentes de atuação (Turmas da ETB), simulando o fluxo real de entrega de valor de grandes corporações.

Turma 2G

Banco de Dados

Foco estratégico na infraestrutura de dados do ETB-PAY. Trabalhamos com SQL avançado, modelagem relacional, arquitetura de extração, transformação e carga (Ciclo ETL) e saneamento rigoroso de bases legadas.

  • Modelagem Lógica e Física
  • Otimização de Queries SQL
  • Pipelines ETL & Saneamento
Turma 3G

LTP II - Java

A engenharia do Back-end. Construímos a lógica de negócios utilizando o paradigma de Orientação a Objetos, integração JDBC, arquitetura limpa e dinâmicas avançadas de Live Coding (Bug Bounty).

  • Padrões POO & SOLID
  • Integração com Banco de Dados
  • Refatoração e Código Limpo
Turma 4F

Gestão de TI

A espinha dorsal operacional. Estudamos a orquestração do sistema através de metodologias Ágeis (Scrum, Kanban) e os frameworks de excelência exigidos pelo mercado público e privado (ITIL, COBIT).

  • Metodologias Ágeis & PMO
  • Framework ITIL 4
  • Governança COBIT
