Meta da Sprint

Construir uma interface Swing de cadastro moderna, conectada de ponta a ponta ao banco de dados corporativo agenciaBancaria.db.

A Infraestrutura do Banco

Antes de construirmos o Formulário Visual (Corpo), precisamos garantir que o Cérebro (Banco de Dados) exista e esteja acessível.

Ação Requerida

Abra o seu terminal SQLite e crie o banco de dados agenciaBancaria.db utilizando os scripts DDL corporativos abaixo antes de iniciar a camada Java.

Prompt Engineering no VS Code

Não precisamos codificar as coordenadas visuais da interface (X, Y) do zero. Vamos utilizar a IA (Gemini) para gerar nosso boilerplate de UI, permitindo que a nossa Squad foque na regra de negócio e integração.

A tela deve conter um painel com fundo escuro (Dark Mode), campos de texto rotulados para Nome, CPF, E-mail, um JPasswordField para Senha e um JButton centralizado escrito 'Cadastrar'. O código não deve usar pacotes (default package)."

A Arquitetura JDBC

O Java não fala nativamente com o SQLite. Precisamos de um tradutor (Driver JDBC) e de uma ponte segura (Connection Factory) para abrir as conexões e tratarmos as possíveis SQLExceptions.

import java.sql.Connection; import java.sql.DriverManager; import java.sql.SQLException; public class ConnectionFactory { public static Connection conectar() { try { // String de conexão apontando para o banco local String url = "jdbc:sqlite:agenciaBancaria.db"; return DriverManager.getConnection(url); } catch (SQLException e) { System.out.println("Erro Crítico de Conexão: " + e.getMessage()); return null; } } }

Padrão DAO & Segurança

Misturar lógica de interface (Swing) com comandos de banco (SQL) resulta no que chamamos de Código Espaguete. O padrão Data Access Object (DAO) isola a comunicação de dados.

Utilizamos a interface PreparedStatement para proteger a nossa aplicação de ataques maliciosos de SQL Injection.

public void salvarCliente(Cliente c) { String sql = "INSERT INTO cliente (nome, cpf, email, senha) VALUES (?, ?, ?, ?)"; try (Connection conn = ConnectionFactory.conectar(); PreparedStatement stmt = conn.prepareStatement(sql)) { stmt.setString(1, c.getNome()); stmt.setString(2, c.getCpf()); stmt.setString(3, c.getEmail()); stmt.setString(4, c.getSenha()); stmt.executeUpdate(); // Executa a gravação física } catch (SQLException e) { System.out.println("Erro ao persistir DAO: " + e.getMessage()); } }
