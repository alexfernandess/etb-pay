Desafio: Setup do Board Estratégico no Jira

A Squad deve configurar o ambiente oficial no Jira Software. O objetivo é estabelecer a visibilidade total da Governança exigida pelo Lean PMO.

GUIA DE CONFIGURAÇÃO: JIRA SOFTWARE

Squad: 4F - Gestão de TI | Projeto: ETB-PAY

1 Criação do Workspace e Projeto

  • Acesse o Jira e crie um ambiente de trabalho (Workspace) chamado etb-workspace:
    https://www.atlassian.com/software/jira
  • Clique em "Criar Projeto" e selecione o template de desenvolvimento de software: Scrum ou Kanban.
  • Defina o tipo de projeto como "Gerenciado pela Empresa".

2 Configuração do Fluxo de Trabalho (Workflow)

O Lean PMO exige as seguintes colunas obrigatórias no Kanban:

TO DO (Backlog)

Demandas aprovadas para a Sprint.

IN PROGRESS

Em desenvolvimento técnico.

CODE REVIEW / QA

Auditoria da Squad 4F - Gestão de TI (Critérios de Aceite e LGPD).

DONE

Em produção, gerando valor.

3 Histórias de Usuário (User Stories) Iniciais

Cadastre as 3 primeiras histórias do MVP:

"1. Como usuário, quero me cadastrar com validação estrita de CPF para evitar fraudes."

"2. Como administrador, quero visualizar transações em um painel central."

"3. Como PMO, quero que a estrutura de banco de dados siga regras de integridade relacional."
